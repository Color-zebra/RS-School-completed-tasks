FSD Architecture
Not a silver bullet!
First level FSD is a layer
Second level FSD is a slice
Public API is an interface that provides an access to a module.
Main purpose - protect and encapsulate.
Segments are the third and final level of the hierarchy.

Should be

  • Layering.
  • Public API.
  • Top-down interaction.

Optional

  • Implementation of all layers.
  • Implementation of slice segmentation.
